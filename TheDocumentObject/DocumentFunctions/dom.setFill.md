# dom.setFill()

[Original Document](http://help.adobe.com/en_US/fireworks/cs/extend/WS5b3ccc516d4fbf351e63e3d1183c94856c-7a3c.html)

## バージョン

3

## 書式

dom.setFill(fill)

## 引数

### fill:

A Fill object (see Fill object). 

Fill型。（Fill型を参照）

## 戻り値

なし

## 説明

Sets the selection to the specified fill.

選択範囲を指定した塗りに設定します。
