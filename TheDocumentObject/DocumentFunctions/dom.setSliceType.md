# dom.setSliceType()

[Original Document](http://help.adobe.com/en_US/fireworks/cs/extend/WSC21840FE-13F1-4d8a-AA5F-E293D4B07798.html)

## バージョン

3、10で変更された

## 書式

dom.setSliceType(slicetype)

## 引数

### slicetype

Specifies the slice type as image (foreground image), background image, or empty (HTML Slice).

画像（全面画像）・背景画像（背景画像）・空（HTMLスライス）からスライスタイプを指定します。

## 戻り値

なし

## 説明

Sets the slice type as image (foreground image), background image, or empty (HTML Slice).

画像（全面画像）・背景画像（背景画像）・空（HTMLスライス）からスライスの種類を指定します。
