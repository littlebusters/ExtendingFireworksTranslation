# dom.setNineScaleGuideColor()

[Original Document](http://help.adobe.com/en_US/fireworks/cs/extend/WS37E5540F-D18E-4949-80BB-2D5ED051F423.html)

## バージョン

9

## 書式

dom.setNineScaleGuideColor(color)

## 引数

### color

Color string as RGB value. 

RSBのString型。

## 戻り値

なし

## 説明

Sets the color that is used to display 9-slice (nonslice) guides.

9スライスガイドの表示で使う色を設定します。
